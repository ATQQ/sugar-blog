## 🔥强烈推荐
### [杀死那个写代码的人](https://windliang.wang/2026/03/31/%E6%9D%80%E6%AD%BB%E9%82%A3%E4%B8%AA%E5%86%99%E4%BB%A3%E7%A0%81%E7%9A%84%E4%BA%BA/)
非常赞同作者的观点👍🏻。

**代码正在从「需要理解的对象」，变成「只要结果正确即可接受的黑盒」。**

* “昨天”：古法写代码过程中，有自己的思考，有debug过程，有发现并解决后的乐趣。
* “今天”：
  * chat 为主，`帮我实现/解决 xxxx`，过去可能手写几个小时的工作量， AI 几十秒就能实现，我们只需要验收符不符合预期。
  * 起初还会 Review 一下，到后来 AI 生成的又多又快，只要功能没问题，代码就直接往上推。
* ”明天“：全链路用 AI 重新塑造，写代码已经是最简单的事了，需要的是新的需求，新的点子，系统要变得 AI 友好。

## 🔧开源工具&技术资讯

### [SSGOI](https://ssgoi.dev/zh) - Native 过度效果
在网页上实现 APP 原生开发的过度效果。

![](https://cdn.upyun.sugarat.top/mdImg/sugar/796df1cccbf0b4cec10810f5aeb1d602)

挺惊艳的，实现 APP 原生开发的过度效果，让基于 Webview 的APP 更像 Native APP。

### [boneyard-js](https://github.com/0xGF/boneyard) - 自动骨架屏

自动生成骨架屏，使用简单，支持多种类型框架。

![](https://cdn.upyun.sugarat.top/mdImg/sugar/306b61776e9af8d01bfa5ce95cc66874)

```vue
<script setup>
import Skeleton from 'boneyard-js/vue'
import './bones/registry'

const loading = ref(true)
</script>

<template>
  <Skeleton name="card" :loading="loading">
    <Card />
  </Skeleton>
</template>
```
### [gitlogue](https://github.com/unhappychoice/gitlogue) - 动画展示 Git 提交历史

动画展示 Git 提交历史，编码的过程可以回放出来。

![](https://cdn.upyun.sugarat.top/mdImg/sugar/ae9d4930451b5f3cc8210902113f7028)

### [gh-space-shooter](https://github.com/czl9707/gh-space-shooter) - 贡献图转为射击小游戏
